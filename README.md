## 📌 Project Title

Telangana PDS Analytics: Multi-Dimensional Shop Performance Clustering and Anomaly Profiling

---

## 🏢 Domain

Public Distribution System (PDS) & Supply Chain Analytics

---

## 🎯 Problem Statement

This project analyzes Fair Price Shops (FPS) in Telangana to identify performance patterns, detect anomalies, and group shops into meaningful clusters for better monitoring and decision-making.

---

## 🚀 Objectives

* Perform data preprocessing and cleaning
* Conduct Exploratory Data Analysis (EDA)
* Engineer meaningful features
* Apply PCA for dimensionality reduction
* Apply K-Means clustering
* Detect anomalies using DBSCAN
* Build an interactive Streamlit dashboard

---
